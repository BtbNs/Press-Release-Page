# Press Release Page

A simple, responsive, and easy-to-deploy frontend template for an organizational news or press release page, designed specifically for hosting on GitHub Pages.

* **`style.css`**: Provides the layout, styling, and basic responsiveness.

### 1. Customization

Before deploying, make sure to customize the following files:

* **`index.html`**:
    * Replace `[Your Organization Name]` with your actual organization's name.
    * Update the placeholder text and metadata within the `<article class="news-card">` sections with your real news content.
    * Update the contact email in the footer.

### 2. Hosting with GitHub Pages

This project is set up for one-click deployment via GitHub Pages:

1.  **Create a Repository**: Create a new public repository on GitHub.
2.  **Upload Files**: Push the `index.html`, `style.css`, and `script.js` files to the **root** of your repository.
3.  **Enable Pages**:
    * Go to **Settings** > **Pages**.
    * Under "Build and deployment", ensure the source is set to **"Deploy from a branch"**.
    * Select your primary branch (`main` or `master`) and the `/ (root)` folder.
    * Click **Save**.

Your site will be live shortly at `https://[Your-Username].github.io/[Repository-Name]/`.
